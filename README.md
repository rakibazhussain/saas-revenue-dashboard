# 📊 SaaS Revenue & Funnel Dashboard

A Power BI dashboard built to analyze SaaS sales pipeline and revenue performance.

![Dashboard](dashboard.png)

## Key Metrics
- **Total Revenue:** $56,153
- **Total Leads:** 500
- **Closed Won Deals:** 47 (9.4% close rate)

## Visuals
- Sales Funnel (Lead → Qualified → Demo → Proposal → Closed)
- Revenue by Sales Rep
- Revenue by Plan Tier (Enterprise / Growth / Starter)

## Tech Stack
- Python (pandas, numpy) — data generation
- Power BI Desktop — dashboard and reporting

## Dataset
Synthetic SaaS sales dataset with 500 records generated using Python.
